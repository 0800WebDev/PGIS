### Description
**PGIS** stands for **Play Games In School** and is an **unblocked games site**
### Official link
[pgis.onrender.com](https://pgis.onrender.com)
### Features
- a cloack option to hide the tab
- an open in about:blank option to hide the page
- contact button to get in touch with the owner
### Apps/tools
Pgis has a selection of tools inclouding:
- orange unlock
- 8os
- live chat
- youtube shorts
- emulatorJS
- AI
### How to use
## Play a game
1. go to the official url
2. click the button saying ">>>MORE GAMES<<<"
3. search or choose a game to play
4. click on the button saying "play" on the game you want to play
## Use a tool
1. go to the official url
2. choose one of the tools from over the logos and below the games and buttons
3. click open
